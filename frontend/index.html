步骤操作说明

数据源管理

当前版本已接入 World Bank、BLS、Eurostat、IMF、OECD、ECB 与 BIS。每个来源都至少有一个可运行的代表性官方指标。

World Bank API

已实现。用于 GDP、GDP 增长率、出口、进口、贸易差额等年度指标。

BLS API

已实现。用于美国 CPI、CPI 同比、失业率、非农就业等月度指标。

Eurostat API

已实现。用于欧元区和欧盟国家 HICP、PPI、工业生产、零售销售、长期利率等月度指标。

IMF

已实现。通过 DataMapper / WEO 接入实际 GDP 增长率等年度指标。

OECD

已实现。通过 SDMX KEI 接入 OECD 国家 CPI 同比等月度指标。

ECB

已实现。通过 ECB Data API 接入欧元兑美元月度平均汇率。

BIS

已实现。通过 BIS Statistics API 接入中央银行政策利率。
